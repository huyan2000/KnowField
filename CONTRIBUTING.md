# Contributing

Thanks for considering a contribution to KnowField.

Good first contributions include:

- adding public field seed configurations under `examples/fields/`;
- improving the field report schema;
- making the paper discovery engine less domain-specific;
- adding tests for metadata parsing and report generation;
- improving documentation for non-expert users.

Please do not contribute copyrighted PDFs, private notes, API keys, local cache files, or generated reports containing personal filesystem paths.
